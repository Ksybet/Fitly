# Fitly

Fitly — мобильное приложение для отслеживания здоровья и активности пользователя.

## 📱 Функциональность (MVP)

### 🔐 Авторизация
- Регистрация пользователя
- Вход (JWT)
- Хранение токена (AsyncStorage)
- Автоматическая проверка сессии

### 👤 Профиль
- Просмотр данных пользователя
- Редактирование профиля (вес, рост и т.д.)

### 🏠 Главный экран (Home)
- Отображение активности за день:
  - Шаги
  - Сон
  - Калории
- Прогресс относительно целей
- Быстрое добавление данных (через карточки)
- Сохранение данных локально (AsyncStorage)

### 🎯 Цели
- Установка целей:
  - Шаги
  - Калории
  - Вес
  - Сон
- Отображение прогресса на главном экране

### 😴 Сон
- Ввод:
  - Время начала и окончания
  - Длительность
  - Качество сна (вручную)
- Отображение на главном экране

## 🛠️ Технологии

### Backend
- Node.js
- Express
- JWT (jsonwebtoken)
- bcryptjs

### Frontend
- React Native (Expo)
- AsyncStorage
- Expo Router
- react-native-svg (графика прогресса)
- Animated API (анимации)
