# Fitly

Fitly — мобильное приложение для отслеживания здоровья и активности пользователя.

## 📱 Функциональность (MVP)

- Авторизация пользователя (JWT)
- Просмотр профиля
- Редактирование профиля

## 🛠️ Технологии

### Backend
- Node.js
- Express
- JWT (jsonwebtoken)
- bcryptjs

### Frontend
- React Native (Expo)

## 🏗️ Архитектура

Mobile App → API → Server

## 🚀 Запуск проекта

### Backend

```bash
cd backend
npm install
npm run dev
